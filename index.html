予算期間設定

会社情報画面に予算開始月を設定し、予算関連画面で一貫した期間表示を実現する

課題: 現状、設定側（月次予算設定）と表示側（予実対比、年間予実対比、月次集計、感度分析）で一貫した予算開始月の設定がない。全画面がカレンダー年（1月〜12月）をハードコードしている。
解決: 会社情報画面に「予算開始月」を追加し、全予算関連画面で参照する。
設定

会社情報画面

担当者メールアドレスの下に「予算期間設定」セクションを追加。予算開始月（1〜12月）をセレクトボックスで設定。

表示

年間予算サマリー

予算開始月を起点に12ヶ月を横並び表示。例: 4月開始なら 4月〜翌3月。

影響

影響を受ける画面

予実対比、年間予実対比、月次集計、感度分析の月選択・期間表示が予算年度に準拠。

# 指定年の予算年度の開始・終了year_monthを返す
def budget_period(year)
  s = budget_start_month || 1
  if s == 1
    { from: year * 100 + 1, to: year * 100 + 12 }
  else
    { from: year * 100 + s, to: (year + 1) * 100 + (s - 1) }
  end
end

# 予算年度の月リスト（表示順）を返す
def budget_months(year)
  s = budget_start_month || 1
  (0..11).map do |offset|
    m = ((s - 1 + offset) % 12) + 1
    y = offset < (13 - s) ? year : year + 1
    { year: y, month: m, year_month: y * 100 + m }
  end
end
