予算期間設定

会社情報画面に予算開始月を設定し、設定機能と表示画面で一貫した予算期間を適用する

課題: 現状、予算の表示期間は「店舗作成日〜現在+3年」の全月リストから手動選択する形式。予算年度（会計年度）の概念がなく、4月始まりや10月始まりなどの年度単位で管理する仕組みがない。
解決: 会社情報画面（会社スコープ）に「予算開始月」を追加し、全店舗の予算関連画面のデフォルト表示期間を予算年度に合わせる。
最新master (2026-04-09) の状態:
- annual_budget_summary.rb: Sorbet型付きにリファクタリング済み。(1..12)ハードコードは解消済み。generate_year_months_listで動的月リスト生成
- monthly_budgets_controller.rb: start_month/end_monthパラメータで任意範囲選択可能
- closing_fiscal_year_settingsテーブルは schema に存在しない（migrate_temp のみ、未デプロイ）

3. Company モデル（Sorbet対応）

# typed: strict
class Company < ApplicationRecord
  validates :budget_start_month, inclusion: { in: 1..12 }, allow_nil: true

  sig { params(year: Integer).returns(T::Hash[Symbol, Integer]) }
  def budget_period(year)
    s = budget_start_month || 1
    if s == 1
      { from: year * 100 + 1, to: year * 100 + 12 }
    else
      { from: year * 100 + s, to: (year + 1) * 100 + (s - 1) }
    end
  end

  sig { params(year: Integer).returns(T::Array[T::Hash[Symbol, T.untyped]]) }
  def budget_months(year)
    s = budget_start_month || 1
    (0..11).map do |offset|
      m = ((s - 1 + offset) % 12) + 1
      y = offset < (13 - s) ? year : year + 1
      { year: y, month: m, year_month: y * 100 + m,
        date: Time.zone.parse("#{y}-#{m}-01") }
    end
  end
end

  end_ym = budget_months.last
